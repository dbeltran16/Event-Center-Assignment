# Elegance Event Center Website

## Project Overview
--
The Elegance Event Center Website is a responsive, multi-page website designed to showcase an upscale event venue.
It highlights venue offerings, upcoming events, and provides a booking/contact form for potential clients.

### The project demonstrates:

- Semantic HTML structure

- Modern CSS layout techniques

- Responsive design principles

- Basic JavaScript interactivity

### Project Structure
--
event-center/
│── index.html        # Home page
│── about.html        # About the venue
│── events.html       # Events & schedule
│── contact.html      # Booking & contact form
│── css/
│   └── style.css     # Main stylesheet
│── js/
│   └── main.js       # JavaScript interactivity
│── images/
│── README.md

### Features Pages
---
Home: Introduction, featured highlights, and gallery

About: Venue history, mission, and team overview

Events: Event listings displayed in a table format

Contact: Booking and inquiry form

### HTML
---
- Semantic elements (header, nav, main, section, article, footer)

- Accessible form elements

- Tables for event schedules

- Internal, external, and anchor links

- Images with descriptive alt text

### CSS Styling
---
- Consistent color palette and typography

- Flexbox and CSS Grid layouts

- Responsive design (mobile-first approach)

- Hover and focus effects

- Custom buttons and transitions

- Background images and gradients

- Class and ID selectors

### JavaScript
---
- Button interaction animations

- Lightweight DOM manipulation

- Enhances user experience without overuse

### Responsive Design
---
The website adapts seamlessly to:

- Desktop screens

- Tablets

- Mobile devices

- Implemented using:

- Media queries

- Flexible layouts

- Scalable images

### How to Run the Project
---
1. Download or clone the project folder

2. Open index.html in any modern web browser (VSC recommneded)

3. Navigate through the site using the menu

4. No server or dependencies required
